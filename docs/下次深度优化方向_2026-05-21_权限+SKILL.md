## P0 · 必做(影响开发可靠性 / 安全边界)

### P0-1 · `bws doctor` 自检命令

**问题**:今天发现的 `~ws_quote-0.8.4.dist-info` 残骸属于"沉默失效",没人手动跑 `bws --help` 就发现不了。

**方案**:加 `bws doctor` 子命令,一次性体检:
- ✅/❌ `pip show bws-quote` 能否找到包 + 版本号是否等于 `pyproject.toml` 声明
- ✅/❌ `site-packages` 里有无 `~*` 前缀的残骸目录
- ✅/❌ `bws.exe` entry-point 能 import `app.cli:main`
- ✅/❌ `alembic_version` 表存在 + 当前 revision = head
- ✅/❌ `.env` 必需字段齐全(ANTHROPIC_API_KEY / DATABASE_URL)
- ✅/❌ tmp DB 可读写(权限)
- ✅/❌ 关键依赖版本(`fastapi>=0.118` `pydantic>=2.x` `alembic>=1.18`)

**位置**:`backend/app/cli/doctor_cmd.py`(新),挂到 `__init__.py` 主 parser

**测试**:`backend/tests/test_cli_doctor.py` — 跑 `bws doctor`,验证退码 0(全绿)/ 1(有红);用 fixture 故意污染 site-packages 验证能检出

**工期**:1-2h(简单 stdlib subprocess + 文件系统检查,无新依赖)

**ROI**:**极高** — 一行命令避免半小时排查;每次升级 pyproject version 后顺手跑一遍就行

### P0-2 · pyproject `version` bump 自动重装 hook

**问题**:今天根因是"改 version 后忘了 `pip install -e .`"。靠人记不可靠。

**方案 A · git hook**(轻量):
- `.githooks/post-commit` 检测本次 commit 是否动了 `pyproject.toml` 的 `version` 行,有就提示用户跑 `pip install -e .`
- 不强制(post-commit 拦不住),只提示

**方案 B · CI 校验**(更严):
- GitHub Actions workflow `.github/workflows/version-consistency.yml`
- 跑 `pip install -e . && python -c "import importlib.metadata as m; assert m.version('bws-quote') == open('pyproject.toml').read()..."`
- 不一致就 fail CI

**方案 C · Makefile target**(最直接):
- `make bump VERSION=0.9.4` → 改 pyproject + 7 处 web 同步 + 自动 pip install -e .
- 把今天 v0.9 业务对齐时改的 7 处版本号写进 sed/python 脚本

**推荐**:**C → B → A** 顺序做。C 立即解决,B 防回归,A 是兜底提示。

**工期**:C 30min + B 1h + A 15min

**ROI**:**高** — 每次发版都会触发,避免"半卸载"再发生

### P0-3 · `.claude/settings.json` 加 `permissions.ask` 中间档

**问题**:目前是 allow / deny 二档:
- allow:静默通过
- deny:硬拦截
缺一档:"我不太确定,**问一下**,但默认 yes"。

`permissions.ask` 就是这第三档:命中 ask 模式会弹询问框,但保留 user override 的机会。

**候选 ask 条目**(从 deny 移上来 / 从 allow 移下来):
- `Bash(git reset --hard *)` — 偶尔需要,但要确认
- `Bash(git branch -D *)` — 同上
- `Bash(git push origin main)` — 主分支推送提示一下
- `Bash(./bws.bat data restore *)` — 覆盖数据库
- `Bash(./bws.bat db migrate *)` — 改 schema
- `Bash(rm -rf *)`(非根目录) — 局部 -rf 也是危险动作
- `Bash(curl http*)` — 非本机网络请求

**工期**:30min(改 settings.json + 测试)

**ROI**:**中高** — 让"95%静默"升级成"99%静默 + 关键节点提醒"

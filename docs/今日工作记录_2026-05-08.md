## 二、技术落地 10 步全程

### 1️⃣ 设计文档 2 份(`docs/`)
- `08_账号权限系统设计.md` — 4 角色矩阵 + 邀请流程 + 密码策略 + 安全 checklist
- `09_ERP同步钩子设计.md` — 队列模式 + payload 协议 + worker 选型 + 监控告警

### 2️⃣ 5 张新表(`models/system.py`)
| 表 | 用途 |
|---|---|
| `agencies` | B 端旅行社主表 |
| `users` | 用户(bcrypt + agency_id + role + 锁定计数) |
| `invitations` | 邀请码 |
| `erp_sync_events` | ERP 同步事件队列 |
| `erp_config` | ERP 配置(enabled/webhook_url/auth_token) |

`Quote` 加 `created_by_user_id` + `agency_id`(ALTER TABLE 兼容)。

### 3️⃣ Auth 重写(`routers/auth.py`)
- bcrypt 哈希(无库时降级 sha256+salt)
- Cookie 升级:`{user_id}.{expiry}.{HMAC-SHA256}` (旧格式不兼容,强制重登)
- 5 次失败锁定 15 分钟
- 老 .env 账号首登自动 bootstrap 成 super_admin(一次性迁移)
- 7 个端点:`/login`、`/logout`、`/me`、`/change-password`、`/register`、`/invitation/{code}`、`/status`

### 4️⃣ 权限辅助(`utils/permissions.py` 新文件)
```python
can_see_costs(user)              # super_admin / agency_owner 可见
filter_quote_dict(d, user)       # 隐藏 cost/profit/gamble
filter_resource_list(rows, rtype, user)  # 隐藏 cost_idr_*
filter_hotel_with_rooms(hotels, user)    # 嵌套 rooms 双层过滤
filter_quotes_by_scope(query, user)      # SQL where 加 agency_id 隔离
require_role(*roles)             # FastAPI Depends 角色装饰器
```

### 5️⃣ ERP 钩子(`utils/erp_hook.py` 新文件)
```python
enqueue_erp_event(db, event_type, entity_type, entity_id, payload):
    cfg = _load_erp_config(db)
    if not cfg.enabled: return None  # 关闭时不写
    db.add(ErpSyncEvent(status="pending"))   # 不 commit
quote_to_payload(quote)  # Quote → ERP webhook JSON
```

### 6️⃣ 管理路由(`routers/admin.py` 新文件,15 个端点合并)
- agencies CRUD(super_admin)
- users CRUD + reset-password(super 全权 / owner 本社)
- invitations CRUD(owner 仅可邀请本社 agent/viewer,越权 403)
- erp-sync 4 端点(events 列表 / mark-synced / skip / retry / config)

### 7️⃣ Quote 改造(`routers/quotes.py`)
- 创建时记录 `created_by_user_id` + `agency_id`
- list/get 按角色 scope 过滤
- 输出 `filter_quote_dict` 裁剪敏感字段
- 新增 PUT `/{id}/status` 状态变 accepted/lost 钉 ERP 事件

### 8️⃣ Gamble feedback 接 ERP(`routers/gamble.py`)
POST `/feedback` 内调 `enqueue_erp_event("gamble.feedback", ...)`

### 9️⃣ Resource 列表全部接字段过滤(`routers/resources.py`)
9 类 list endpoint(hotels / attractions / restaurants / vehicles / guides / optional-tours / spa / water / tea)return 前接 `filter_resource_list(rows, rtype, user)`。

## 三、用户操作示范(回应原话)

> "我（公司管理员）想给上海康辉开账号"

```
1. /admin → 旅行社管理 → + 新增 → 填名字 "上海康辉"
2. 邀请码 → + 生成 → role=agency_owner / 7 天有效
3. 复制链接发给王总: https://yourdomain.com/register?code=bws-XXXXX
4. 王总点链接 → 设用户名密码 → 自动登录成 agency_owner
```

> "我（B 端老板）想给手下小李开账号"

```
1. 我的团队 → + 邀请新成员 → role=agent / 30 天 / 备注: 给小李用
2. 复制链接发小李
3. 小李注册后:
   ✓ 看到本社所有资源(无 IDR 成本)
   ✓ 自己创建的 quote 看得到 ¥售价(无 cost/profit/gamble)
   ✓ 看不到其他业务员或其他社的报价
```

> "成单了,要推 ERP"

```
v0.4 暂时:
- super_admin 改 quote.status=accepted
- 队列默认不写(enabled=False)
- 想用时 PUT /erp-sync/config → enabled=true
- 启用后 quote.status 变 accepted/lost 自动写事件
- super_admin 在 /erp-sync/events 看队列 → 手动 mark-synced

v0.5 真上 worker 后:
- 队列自动 POST 到 ERP webhook → ERP 端 UPSERT 创建订单
- 失败 5 次 → 告警 + 人工干预
```

**做出来的东西**（一句话版）：
新增 `GambleStrategy` 单表 + Settings 加 "💰 赌自费策略" 卡片 + "🔍 案例编辑器" 预览器。每条策略 = 一种行程组合 + 对应让利金额（不赌 / 固定 ¥/人 / 按团 ¥/团）。引擎按 priority 倒序评估，首条命中即停。

## 三、用户案例（最直观的回应）

> "我家庭团 + 半天自由想让 ¥250 而不是默认 ¥200，怎么办？"

**操作 4 步**：
1. Settings Tab → 💰 赌自费策略
2. 点"+ 新增策略" → 填：
   - 名："家庭半天自由 让 ¥250"
   - 动作：固定让利
   - 金额：250
   - 优先级：55（高于默认兜底的 1）
   - 条件：customer_type_in=family AND free_hours_gt=3 AND free_hours_lt=8
3. 保存 ✓
4. 想验证？点"🔍 案例编辑器/预览" → 填 customer_type=family / free_hours_total=5 → 评估 → 看到命中你刚建的 ¥250 策略 ✓

**整个交互 0 公式 0 系数**，就是"哪种 → 多少"的人话。
